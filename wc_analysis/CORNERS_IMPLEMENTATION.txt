## 1. 实现概述

在 predict.py 中新增 `get_corner_boost()` 函数,按优先级提供角球能力加成:
1. **corners.json 手动数据** (优先级最高)
2. **Elo 等级估算** (fallback)

加成范围: 0-0.15,直接乘到 λ 上(如 λ × (1 + boost))

## 2. 代码修改

### 2.1 添加配置文件路径
```python
CORNERS_FILE = DATA_DIR / "corners.json"
```

### 2.2 实现 get_corner_boost() 函数
```python
def get_corner_boost(team: str) -> float:
    # 1. 优先读 corners.json
    if CORNERS_FILE.exists():
        corners_data = json.loads(...)
        if team in corners_data:
            cpg = corners_per_game  # 场均角球数
            ctg = corners_to_goals   # 角球转化率
            # 综合评分: cpg/7 × 0.5 + ctg/0.12 × 0.5, 上限 0.15
            score = (min(cpg/7, 1) * 0.5 + min(ctg/0.12, 1) * 0.5) * 0.15
            return score

    # 2. fallback: 用 Elo 估算
    elo = get_elo(team)
    if elo >= 2000:
        boost = 0.12
    elif elo >= 1500:
        boost = (elo - 1500) / 500 * 0.12
    else:
        boost = 0.0
    return min(boost, 0.15)
```

### 2.3 集成到 get_adjustments()
```python
def get_adjustments(home: str, away: str) -> tuple[float, float, list[str]]:
    adj_h, adj_a = 1.0, 1.0
    notes = []

    # 角球/定位球能力加成
    corner_boost_h = get_corner_boost(home)
    corner_boost_a = get_corner_boost(away)
    if corner_boost_h > 0.02:
        adj_h *= (1.0 + corner_boost_h)
        notes.append(f"{home}定位球能力强(λ×{1+corner_boost_h:.3f})")
    if corner_boost_a > 0.02:
        adj_a *= (1.0 + corner_boost_a)
        notes.append(f"{away}定位球能力强(λ×{1+corner_boost_a:.3f})")

    # ... 原有的风格相克、磨合度、伤病、天气调整继续执行
```
